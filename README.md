# Tailwind CSS Button Component

This project contains a comprehensive set of Tailwind CSS button components with pure CSS styling. The buttons are designed to be reusable and customizable for various use cases.

## Features

- Multiple button variants (primary, secondary, success, danger, warning, info)
- Outline button styles
- Different sizes (small, normal, large)
- Special button styles (pill, ghost, link)
- Disabled state styling
- Hover, focus, and active states
- Responsive design

## Usage

To use these button components, simply include the CSS file in your project and add the appropriate classes to your button elements:

```html
<button class="btn btn-primary">Primary Button</button>
<button class="btn btn-secondary btn-lg">Large Secondary Button</button>
<button class="btn btn-outline btn-outline-success">Outline Success Button</button>
```

## Button Classes

### Base Class
- `btn` - Base styling for all buttons

### Variants
- `btn-primary` - Primary button style
- `btn-secondary` - Secondary button style
- `btn-success` - Success button style
- `btn-danger` - Danger button style
- `btn-warning` - Warning button style
- `btn-info` - Info button style

### Outline Variants
- `btn-outline-primary` - Primary outline button
- `btn-outline-secondary` - Secondary outline button
- `btn-outline-success` - Success outline button
- `btn-outline-danger` - Danger outline button
- `btn-outline-warning` - Warning outline button
- `btn-outline-info` - Info outline button

### Sizes
- `btn-sm` - Small button
- `btn-lg` - Large button

### Special Styles
- `btn-pill` - Pill-shaped button
- `btn-ghost` - Ghost button style
- `btn-link` - Link-style button

### States
- Disabled: Add the `disabled` attribute to the button element
